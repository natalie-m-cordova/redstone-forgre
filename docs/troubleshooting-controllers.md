# Troubleshooting – Controller (Windows Client)

This guide helps fix common problems when using Redstone Forge from a Windows PC.

These steps are designed to be simple and quick.

---

# 1. Cannot Open the Control Panel

## Problem

The page does not load when you go to:
```
http://<server-ip>:8000
```

# When to Ask for Help

Ask for help if:

- The page won’t load at all
- The server will not start or stop
- Minecraft cannot connect after multiple tries
- Uploads consistently fail

---

# Summary

Most issues can be fixed by:

- Refreshing the page
- Checking the server is running
- Making sure you are on the same network
